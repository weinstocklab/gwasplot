# Search for available cell types and their biosample IDs

Search for available cell types and their biosample IDs

## Usage

``` r
search_cell_types(search_term = NULL, assembly = "grch38")
```

## Arguments

- search_term:

  Optional search term to filter cell types (case-insensitive)

- assembly:

  Genome assembly ("grch38" or "mm10")

## Value

Data frame with cell type information including biosample IDs
