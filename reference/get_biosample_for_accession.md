# Get biosample to accession mapping for a specific accession

Get biosample to accession mapping for a specific accession

## Usage

``` r
get_biosample_for_accession(accession, assembly = "grch38")
```

## Arguments

- accession:

  cCRE accession number (e.g., "EH38E1234567")

- assembly:

  Genome assembly ("grch38" or "mm10")

## Value

Data frame mapping biosample IDs to this accession
